### Math stuff

An estimator $\hat{\theta}$ is consistent for $\theta$ if
$$
\hat{\theta}\stackrel{p}{\to}\theta,
$$
i.e., if $P(|\hat{\theta}-\theta|>\epsilon)\to0$ for every $\epsilon>0$.

Minimum requirement for a useful estimator.
